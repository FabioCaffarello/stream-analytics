# Doc Contract Template

**Status:** Active
**Owner:** Governance Doc-First Maintainer
**Last updated:** 2026-02-13

## ADR Template (short)

- Header:
- `Status`
- `Owner`
- `Last updated`
- `Date`
- `Deciders`
- `Relates to`
- Required sections:
- `Contexto`
- `Decisao`
- `Consequencias`
- `Invariantes`
- `Evidence`
- `Changelog`
- Conditional rule:
- se parcial, usar `Status: Partially Implemented` + `Implementation Matrix`

## RFC Template (short)

- Header:
- `Status`
- `Owner`
- `Last updated`
- `Date`
- `Author`
- `Relates to`
- Required sections:
- `Objetivo`
- `Escopo`
- `Nao-Escopo`
- `Design`
- `Rollout`
- `Test Plan`
- `Acceptance`
- `Risks`
- `Evidence`
- `Changelog`
- Conditional rule:
- se parcial, usar `Status: Partially Implemented` + `Implementation Matrix`
