## Problem

Define the durable competitive advantages that make Stream Analytics difficult to replicate:
sub-millisecond multi-venue normalization, deterministic replay, actor-supervised reliability,
and the integrated Odin cockpit as a single coherent operator surface.

## Proposed Moat Dimensions

1. **Data fidelity** — canonical multi-venue normalization with gap detection and deterministic replay
2. **Latency** — p50=7µs, p95=13µs pipeline latency at 117K evt/sec (C4 soak baseline)
3. **Reliability surface** — 5-layer stream health pipeline with operator-visible signals
4. **Integrated cockpit** — 13 widget types, 8 indicators, WASM + native client
